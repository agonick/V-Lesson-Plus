# Code of Conduct

## Our Pledge

We want this project to be welcoming, respectful, and inclusive for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent and discuss ideas, not people.
- Accept feedback gracefully.
- Help create a positive collaboration environment.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Racist language or behavior.
- Discriminatory language or behavior.
- Trolling, intimidation, or sustained disruption.
- Sharing private information without consent.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct.

## Reporting

Report abusive behavior to: agonick-dev@proton.me

All reports will be reviewed and handled fairly.
