# Security Policy

## Supported Versions

This is an early-stage project. Security fixes are applied to the latest code on the default branch.

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities.

Report privately by email:

- agonick-dev@proton.me

Include:

- A clear description of the issue
- Steps to reproduce
- Impact assessment
- Suggested fix (if available)

## Response Targets

- Initial acknowledgment: within 7 days
- Triage/update: as soon as feasible

## Scope

Focus reports on:

- Unsafe extension permissions
- Message-passing vulnerabilities
- Data leakage in storage/logging
- Supply-chain risk in dependencies
